# rn-tpm-gateway-lib

NPM pkg for access the native ios/android TPM libs for payment

## Installation

```sh
npm install rn-tpm-gateway-lib
```

## Usage

```js
import { multiply } from 'rn-tpm-gateway-lib';

// ...

const result = await multiply(3, 7);
```
